Understanding Networking and Server Systems

Wi-Fi USB Dongles

A Wi-Fi USB dongle is a small device that plugs into a computer's USB port to provide wireless network connectivity

3G routers allow to connect USB dongle to the router just like how we connect cable connection to router. User can into connect USB dongle as internet connection to the router and use the internet over Wi-Fi on multiple devices.

"Not every USB router compatible with USB dongle internet connection."

If you are planning to buy a router to use with USB dongle and cable internet connection, please look for terms like 3G router, USB dongle in product specification. Normally USB port in the router is designed for connecting external hard disk or printer. In this type USB port USB dongle won't work. So be careful while purchase router in which you want connect 3G USB dongle. Look for "dongle" or "USB dongle" phrase in product specification or product reviews.

Types of Networking and Wireless Connections

V/S

Networking refers to connecting computers and devices to share resources and communicate. It can be wired or wireless, using various technologies like Ethernet, Wi-Fi, and Bluetooth.

Common types of networks include:

  • LAN (Local Area Network)
  • WAN (Wide Area Network)
  • Hotspots

Wireless networking allows devices to connect without physical cables. Instead, data is transmitted using radio waves or infrared.

Wireless networks are vulnerable to unauthorized access if not secured. Common security protocols:

  • WEP (outdated)
  • WPA / WPA2 / WPA3 (modern, more secure)

Best practices:

  • Use strong passwords
  • Enable WPA3 encryption if possible
  • Keep firmware up to date

Common Functionality of Server Networks

Servers manage resources and data across networks. They perform tasks like hosting websites, storing files, and running applications.

Common server types include:

1. File Sharing & Storage

  • Servers store and manage files for multiple users.
  • Allows centralized access, backups, and permissions.
  • Example: File Server

2. Print Services

  • Manages shared printers across the network.
  • Users can send print jobs from any connected device.
  • Example: Print Server

3. User Authentication & Authorization

  • Controls who can access the network and what they can do.
  • Uses Active Directory or similar systems.
  • Example: Domain Controller

4. Web Hosting

  • Hosts websites and web applications.
  • Provides services over HTTP/HTTPS.
  • Example: Web Server (Apache, Nginx, IIS)
